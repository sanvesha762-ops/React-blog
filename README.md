# React Blog - Bollywood Edition

A clean, responsive React blog featuring the latest stories from the Bollywood industry.

## Features
- **Dynamic Home Page**: Displays featured stories from the collection.
- **Category Based Filtering**: Specialized page for Bollywood content.
- **Modern UI**: Built with React, Vite, and custom CSS with dark mode support.
- **Responsive Design**: Optimized for mobile, tablet, and desktop views.

## Project Structure
- `vite-project/`: The main React application.
- `src/Components/`: Contains Home, Bollywood, and Navbar components.
- `src/Data.js`: Centralized data store for blog posts.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sanvesha762-ops/React-blog.git
   ```
2. Navigate to the project directory:
   ```bash
   cd React-blog/vite-project
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server:
```bash
npm run dev
```
